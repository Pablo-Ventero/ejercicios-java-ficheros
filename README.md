# ☕ Ejercicios Java — Ficheros de Texto

Colección de proyectos en Java trabajando lectura y escritura de ficheros
desarrollados durante el primer curso del ciclo formativo DAM.

## 📋 Proyectos

| # | Proyecto | Conceptos |
|---|----------|-----------|
| 01 | [Traductor a Código Morse](./01-traductor-morse/) | Leer/escribir ficheros, arrays, búsqueda binaria |
| 02 | [Contador de texto](./02-contador-texto/) | Leer fichero, contar palabras, párrafos y caracteres |
| 03 | [Filtro de productos CSV](./03-filtro-productos-csv/) | Leer CSV, filtrar por categoría, escribir resultado |
| 04 | [Filtro de empleados CSV](./04-filtro-empleados-csv/) | Leer CSV, filtrar por departamento, calcular media |
| 05 | [Filtro de alumnos CSV](./05-filtro-alumnos-csv/) | Leer CSV, filtrar por ciudad y nota, escribir resultado |

## ▶️ Cómo ejecutar cualquier proyecto

**Requisitos:** tener Java instalado y el fichero de datos en la misma carpeta.
```bash
# Entrar a la carpeta del proyecto
cd 01-traductor-morse

# Compilar
javac Principal.java

# Ejecutar
java Principal
```

## 📚 Conceptos cubiertos
- Lectura de ficheros con `BufferedReader` y `FileReader`
- Escritura de ficheros con `BufferedWriter` y `FileWriter`
- Try-with-resources para cerrar ficheros automáticamente
- Procesamiento de ficheros CSV con `split(",")`
- Búsqueda binaria con `Arrays.binarySearch()`
- Filtrado y transformación de datos entre ficheros
